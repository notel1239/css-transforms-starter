# CSS Transformations Learning Lab

Welcome to the CSS Transformations Learning Lab! This repository is designed to help you master the fundamentals and some intermediate concepts of CSS 2D and 3D transformations through a hands-on, GitHub-driven workflow.

## Overview

This lab consists of five lessons, each focusing on a different transformation concept. You'll learn by doing, using GitHub Issues to track your progress, branches to develop each concept, and Pull Requests to undergo interactive code reviews.

## Repository Structure

- `README.md` - This file
- `lessons/` - Directory containing markdown lesson files (lesson-01.md through lesson-05.md)
- `index.html` - The main HTML file where you'll implement transformation examples
- `css/styles.css` - The stylesheet where you'll write your CSS transformations
- `.github/pull_request_template.md` - Template for your pull request descriptions

## How to Use This Lab

1. **Fork this repository** (if you haven't already)
2. **Clone your fork** to your local machine
3. Follow the workflow described in each lesson:
   - Create an issue from the lesson content
   - Create a feature branch
   - Implement the transformation
   - Open a pull request
   - Review the interactive comments
   - Merge and close the issue

## Lessons

1. **Lesson 01: 2D Rotation** - Learn how to rotate elements in 2D space
2. **Lesson 02: Scaling** - Understand how to scale elements up and down
3. **Lesson 03: Translation** - Move elements around the page
4. **Lesson 04: 3D Transformations** - Add depth with perspective and 3D rotations
5. **Lesson 05: Combining Transforms** - Chain multiple transformations together

## Prerequisites

- Basic knowledge of HTML and CSS
- Familiarity with Git and GitHub
- A modern web browser that supports CSS transforms

## Getting Started

Start with Lesson 1 by checking the `lessons/lesson-01.md` file and following the instructions there.

Happy transforming!
